## 关键变化
- `run.ps1` 新增：`-Protocol json|raw`

## JSON Envelope 结构
```json
{
  "status": "ok|error",
  "intent": "track",
  "timestamp": "2026-05-12T11:21:10",
  "exit_code": 0,
  "data": {
    "script": "opl_track.ps1",
    "params": {"ConfigPath": "..\\config\\skill_config.json"},
    "parsed_json": null,
    "output_text": "...",
    "output_lines": ["..."]
  },
  "errors": []
}
```

## 兼容性说明
- 对模型：默认用 `-Protocol json`，最稳定
- 对人工调试：可用 `-Protocol raw`
- 即使参数缺失（例如 analyze 无 Query），json 模式也会返回统一错误结构
